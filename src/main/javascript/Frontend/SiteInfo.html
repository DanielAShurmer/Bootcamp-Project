Bug Tracker

This website is a bug tracking system - it was designed to be deliberately generic, so that it could be used as a template and adapted in order to work as a bug monitoring system for any computer application or videogame.

This site was created as part of a programming bootcamp project, in order to demonstate my (Daniel Shurmer) knowledge of front-end / back-end website coding and how a database can be interacted with via a website. It is by no means secure, and no data you don't want other people to see or potentually delete should be stored using it.
